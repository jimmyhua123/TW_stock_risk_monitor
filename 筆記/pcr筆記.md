# PCR (Put/Call Ratio) 趨勢觀察筆記

## 1. 觀察背景
在 **1月至2月初** 期間，透過 **未平倉量比率 % (P/C Ratio)** 觀察到市場發生了顯著的「趨勢變化」。
主要探討：為什麼 **1/29、1/30、2/2** 會下跌？

## 2. 數據追蹤

| 日期 (Date) | P/C Ratio (未平倉量比率) | 狀態備註 | Buy/Sell Volume Ratio (買賣權成交量比率) |
| :--- | :--- | :--- | :--- |
| **1/28** | **191.87%** | **極度看多** | 106.18% |
| **1/29** | **134.57%** | **暴跌 57%** | 86.58% |
| **1/30** | **157.04%** | 稍回升 | 89.42% |
| **2/2** | **109.20%** | **再次大減** | 81.40% |
| **2/3** | **131.55%** | 稍微回穩 | 116.58% |

### 關鍵訊號解讀
> **訊號：** 當 P/C Ratio 從高位快速下滑（例如從 191% 掉到 134%）

1.  **大戶動向**：這代表大戶正在**撤退**或**反手佈空**。
2.  **市場支撐**：雖然數值仍維持在 100% 以上（代表多方仍佔優），但**「支撐力道正在快速消失」**。
3.  **後市預警**：這種劇烈的數值修正，往往是行情**轉折**或即將**回檔**的強烈先兆。

## 3. 深入解析：PCR 指標的迷思與正確用法

### 1) 為什麼 Put/Call 比率 >100% 不一定看空？
**核心原因：未平倉量 (OI) 與 成交量 (Volume) 都不分多空方向。**

OI / Volume 只是在數合約數量，不告訴你這些 Put 是「買進」還是「賣出」。同一筆 Put 合約，必然同時存在買方與賣方。

所以「Put 變多」可能是：
*   **大量買 Put**（偏空/避險）
*   **大量賣 Put**（偏多/收權利金）
*   或是兩者都有，但誰主導需要更多資訊才能判斷。

這就是你看到「比率高但行情仍漲」的第一個關鍵：**Put 多不等於看空，多數時候它可能是避險或賣 Put 的結構。**

### 2) 台股（常見口徑）為何容易長期出現 >100%？
下面是幾個「結構性」理由，任何一個成立都會讓比率偏高，但不必然壓制行情：

*   **(A) 上漲行情中，現股多單常用 Put 當保險（避險盤）**
    *   很多資金是長期持股/指數多單，同時買 Put 做保險。
    *   行情漲不代表保險會被取消，反而可能一直續保 → **Put OI 偏高但現貨仍上漲**。
*   **(B) 很多人用「賣 Put」收權利金（偏多策略）**
    *   賣 Put 的本質：看法偏「不跌/盤整偏多」或願意接刀。
    *   這會造成 Put OI/成交量上升，但它其實是偏多的部位結構。
    *   只看 Put/Call 比率會被誤導成「偏空」。
*   **(C) 深度價外 Put（遠 OTM）便宜、常被當尾端風險保險**
    *   便宜的災難保險 Put 可能大量存在。
    *   這類 Put 的量可以很大，卻不會每天影響現貨趨勢。
*   **(D) 統計口徑問題**
    *   Put 通常分布在更廣的履約價帶、也更常被拿來做保護，合併後比率偏高是常見現象。

### 3) 關於「補保證金砸盤」假說的驗證
你提出「補保證金砸盤」的可能性，若此說法成立，通常會看到以下連動特徵：
1.  指數/個股出現**急跌**。
2.  同時 Put 成交量（或 PCR-Volume）**急升**。
3.  波動率（IV/VIX 類）**同步急升**。
4.  期貨/融資維持率/追繳等資訊出現壓力訊號。

**結論：**
你描述的是「1 月整月大漲、只有少數日下跌」，這種型態更像 **結構性避險/賣 Put 收權利金長期存在**，而不是「被追繳保證金導致砸盤」。

### 4) 那 PCR 要怎麼用才不會被騙？

#### ✅ 正確用法 1：看「極端」與「變化」，不要只看 >100
*   絕對值 >100 在資料裡常是「常態區」。
*   你要看的是：
    *   是否出現**「相對自身」的極端**（例如接近近 1~3 個月高點）。
    *   是否在短期內**快速跳升（ΔPCR）**。

#### ✅ 正確用法 2：用 OI 和 Volume 分工
*   **PCR-OI**：偏**「中期結構」**
*   **PCR-Volume**：偏**「當日情緒」**

| 情境 | 解讀 |
| :--- | :--- |
| **OI 長期高但 Volume 沒有飆** | 結構性因素（避險/賣 Put） |
| **Volume 突然飆高且價格大跌** | 當日恐慌/事件衝擊 |

#### ✅ 正確用法 3：一定要搭配「價格與波動」判讀
*   **PCR 高 + 指數上漲 + 波動不升**：通常**不是**恐慌空。
*   **PCR 高 + 指數急跌 + 波動急升**：才更像**風險事件**。
