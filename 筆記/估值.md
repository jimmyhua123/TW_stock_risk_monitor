# 估值指標筆記

## 1. 本益比 (P/E) —— 適用於「獲利穩定」的公司

- **定義**：股價 / 每股盈餘 (EPS)。代表「你需要幾年才能回本」。
- **盲點**：
  - **看的是過去**：一般的本益比是用「去年」或「近四季」的 EPS 計算。但股價是反映未來，所以法人更看重**「預估本益比」(Forward P/E)**，也就是用「預測未來的 EPS」來算。
  - **一次性獲利**：如果公司賣土地賺了一大筆錢，EPS 暴增，本益比瞬間變很低（例如剩 5 倍），這時買進通常會賠錢，因為明年就沒有這筆收入了。
- **如何判斷昂貴？** 通常與「同業」比較，或與「該公司過去 5 年的區間」比較。

## 2. 股價淨值比 (P/B) —— 適用於「景氣循環股」

- **定義**：股價 / 每股淨值。代表「股價是公司清算價值的幾倍」。
- **適用對象**：獲利起伏劇烈，但資產價值穩定的產業。例如：航運、鋼鐵、面板、記憶體。
- **為什麼不用本益比？** 當景氣不好時，這些公司可能賠錢（EPS 為負），本益比算不出來或無限大；但它們的工廠、設備（淨值）還在。
- **判斷邏輯**：
  - 當 P/B 跌到 1 倍以下（甚至歷史低點）通常是長線買點。
  - 當 P/B 過高（例如超過 2 倍以上）則需警惕。

## 3. 本益成長比 (PEG) —— 適用於「高速成長股」

- **定義**：本益比 / 盈餘成長率。這是投資大師彼得·林區（Peter Lynch）推崇的指標。
- **適用對象**：AI、IC 設計、矽光子等高成長科技股。
- **判斷邏輯**：一家公司本益比 30 倍看起來很貴？但如果它每年獲利成長 40%，那麼 $30 / 40 = 0.75$。
- **標準**：
  - PEG < 1 代表股價被低估（成長速度大於估值）。
  - PEG > 1.5 則可能開始過熱。
  - 這解釋了為什麼某些科技股本益比高達 50 倍，股價卻還一直漲，因為市場預期它的成長率極高。

## 綜合判斷表

| 指標 | 適用產業/情境 | 判斷過熱的警訊 (參考) |
| :--- | :--- | :--- |
| **本益比 (P/E)** | 食品、電信、代工 (獲利穩健型) | 顯著高於歷史平均區間 (如平時 12 倍，突然變 20 倍) |
| **淨值比 (P/B)** | 記憶體、面板、航運 (景氣循環型) | P/B 來到歷史高緣 (如航運股 P/B > 3) |
| **本益成長比 (PEG)** | AI、半導體先進製程 (高成長型) | PEG > 1.5 或 2 (股價漲幅遠超獲利成長速度) |

## 結論

判斷估值不能「一招鮮吃遍天」。

- 如果是看 **台積電** 或 **IP 股**（成長型），要看 **PEG** 或 **預估本益比**。
- 如果是看 **面板** 或 **貨櫃輪**（循環型），請務必切換成 **股價淨值比 (P/B)**。

否則很容易在股價最高點（獲利最好、本益比最低時）進場接刀。
